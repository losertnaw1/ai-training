# Stable Diffusion Training with LAION-400M

This project implements a training pipeline for fine-tuning Stable Diffusion models using the LAION-400M dataset, a large-scale dataset containing 400 million image-text pairs.

## Project Structure

- `src/`: Source code for the project
- `data/`: Data handling and preprocessing
- `models/`: Model implementations and saved checkpoints
- `configs/`: Configuration files
- `notebooks/`: Jupyter notebooks for exploration and visualization
- `scripts/`: Utility scripts
- `docs/`: Documentation

## Setup

1. Create a virtual environment
2. Install dependencies: `pip install -r requirements.txt`
3. Configure environment variables

## Usage

See the documentation in the `docs/` directory for detailed usage instructions.
